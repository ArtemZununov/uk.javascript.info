
Зворотні лапки вбудовують вираз, що знаходиться всередині `${...}`, у рядок.

```js run
let name = "Ілля";

// вираз — число 1
alert( `привіт ${1}` ); // привіт 1

// вираз є рядком "name"
alert( `привіт ${"name"}` ); // привіт name

// вираз є змінною, яка вбудовується
alert( `привіт ${name}` ); // привіт Ілля
```
